Grid Layout

O que é Grid Layout?

É uma especificação projetada criação de layout através de método bi-dimensional, possibilitando organizar os itens da página co o uso de linhas e colunas. O diferencial desse método é que ele possibilita dividir uma página em grandes regiões e definir o relacionamento em termos de medidas, posicionamento e camadas entre os diferentes componentes de marcação HTML.

Grid Container

É a tag que envolve os itens do grid, ao indicar display:grid, essa tag passa a ser um Grid Container.

.container {display:grid}

grid-template-colums e grid-template-rows

Como estamos trabalhando com grade, é necessário indicar quantas linhas e quantas colunas ela irá ter. Para colunas usamos o grid-template-colums e para linhas usamos o grid-template-rows

O valor dado ao número de colunas ou linhas que queremos é indicado para cada uma. Por exemplo, se queremos 4 colunas, não colacaríamos o número quatro, mas sim a medida de cada, em qualquer unidade (px, em, %, cm etc), separada por um espaço.

grid-template-columns: 20% 5rem 40%; grid-template-rows: 40% 60%;

grid-template-areas

O CSS Grid permite ir além de apenas organizar a grid de acordo com a ordem em que estão no HTML. Podemos dar nomes para cada área da grade e depois indicar onde cada elemento deve ir. Esse modo deixa bem simple de entender a estrutura do grid e é feito através da propriedade grid-template-areas

.area-container { display: grid; grid-template-columns: 40% 60%; grid-template-rows: 30% 50% 20%; grid-template-areas: 'header header' 'sidenav content' 'footer footer';}

.header { grid-area: header; } .sidenav { grid-area: sidenav; } .content { grid-area: content; } .footer { grid-area: footer; }

Grid Item

Os Grid Itens são os filhos diretos do Grid Container. Um grid item pode ser explícito ou implícito. Explicito é quanto você define ele explicitamente no container e implícito é quanto ele é criado automaticamente para preencher o conteúdo no grid.

grid-column

Define quais colunas serão ocupadas pelo grid item. É possível definir uma linha de início e final, assim o item irá ocupar múltiplas colunas.

  • grid-column: 1- O item ocupará a coluna 1.
  • grid-column: 1 / 3- O item ocupará a coluna 1 e 2 (Sim, isso mesmo, 1 e 2, pois os valores 1 / 3 são referentes as linhas da coluna. Isso significa que começa na linha 1 (início do grid) e vai até a linha 3, que é o começo da terceira coluna).
  • grid-column-start: 2- item vai começar na linha 2.
  • grid-column-end: 4- O item vai terminar na linha 4.
  • grid-column: span 2- O item irá ocupar duas colunas a partir de onde ele estiver.

grid-row

Define quais linhas serão ocupadas pelo grid item. Essa linha é referente a row. Porém as chamadas grid lines que por tradução também significam linhas do grid, são diferentes. Uma row (linha), possui sempre 2 grid lines (linhas do grid), uma no início dela e uma no final dela.

  • grid-row: 1 - O item ocupará a linha 1.
  • grid-row: 1 / 3- O item ocupará a linha 1 e 2 (Sim, isso mesmo, 1 e 2, pois os valores 1 / 3 são referentes as linhas do grid. Isso significa que começa na linha 1 (início do grid) e vai até a linha 3 do grid, que é o começo da terceira linha).
  • grid-row-start: 2- O item vai começar na linha do grid 2.
  • grid-row-end: 4- O item vai terminar na linha do grid 4.
  • grid-row: span 2- O item irá ocupar duas linhas a partir de onde ele estiver.

O que é unidade fr?

A unidade fr representa uma fração do espaço disponível no container do grid. Dessa maneira, ela contribui para a criação de layouts flexíveis.

.container { display: grid; grid-template-columns: 2fr 1fr 1fr; height: 7rem; border: 2px solid red; padding: 1rem; }

O que é a noção de repeat?

Grids grandes, com muitas tracks podem usar a notação repeat() para repetir todas ou uma seção apenas.

Veja o exemplo abaixo: Ao invés de colocar grid-template-columns: 1fr 1fr 1fr 1fr 1fr;, coloca-se o repeat afim de facilitar o código:

.container-repeat { display: grid; grid-template-columns: repeat(5, 1fr); grid-template-rows: repeat(2, 1fr);}

O que são as linhas do Grid?

Ao definir um grid, designamos as trilhas dos mesmo, não as linhas. No entanto, o próprio grid retorna linhas numeradas para serem usadas ao posicionar itens. As linhas entre as columns são denominadas de column lines e as linhas entre rows são denominadas row lines.

No exemplo abaixo está posicionado os primeiros itens do grid de 3 colunas, usando as propriedades grid-column-start,grid-column-end,grid-row-start e grid-row-end.

Da esquerda para a direita, a primeira box é posicionado contra a linha da coluna 1, e vai até a linha da coluna 4, que no caso é a linha do extremo direito do grid. Começa na linha 1 e termina na linha 3, tendo uma expansão de duas linhas.

A segunda box inicia na linha da coluna 1, e expande uma linha. Esse é o padrão, então não precisa especificar a linha final. Também vai da linha 3 a linha 5. Os outros itens serão posicionados de acordo com espaço disponível no grid.

.wrapper { display: grid; grid-template-columns: repeat(3, 1fr); grid-auto-rows: 100px; } .box1 { grid-column-start: 1; grid-column-end: 4; grid-row-start: 1; grid-row-end: 3; } .box2 { grid-column-start: 1; grid-row-start: 3; grid-row-end: 5; }

O que são áreas do Grid?

Os Grid Items podem se espalhar por uma ou mais células ambas entre linhas e colunas, e isto cria uma Área de Grid. As áreas de grid devem ser retangulares, ou seja, não é possível criar uma área em L por exemplo.

Basicamente define a área do item do grid. É um atalho para grid-row-start, grid-column-start, grid-row-end, grid-column-end.

grid-area: 1 / 2 / 4 / 3; // É um atalho para: grid-row-start: 1; grid-column-start: 2; grid-row-end: 4; grid-column-end: 3;

Abaixo o exemplo:

#grid { display: grid; grid-template-columns: repeat(4, 1fr); } #item1 { grid-area: 2 / 2 / auto / span 3; }
